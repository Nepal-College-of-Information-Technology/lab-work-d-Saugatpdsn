## Background Theory

# Role of Cloud Computing in IoT Systems

## Introduction

The *Internet of Things (IoT)* refers to a network of connected devices such as sensors, smart appliances, wearable devices, and machines that communicate through the internet.
Cloud computing provides the infrastructure required to store, process, and manage the large amount of data generated by IoT devices.

---

## Roles of Cloud Computing in IoT

### 1. Data Storage

Cloud platforms provide large-scale storage systems for IoT data.

* Stores sensor and device data
* Provides backup and recovery
* Supports centralized data management

---

### 2. Data Processing

IoT devices generate huge amounts of data that require processing and analysis.

Cloud computing helps by:

* Performing real-time data analysis
* Running AI and Machine Learning algorithms
* Detecting patterns and making predictions

---

### 3. Remote Access and Monitoring

Cloud services allow users to control and monitor IoT devices from anywhere.

#### Examples

* Smart home control systems
* Remote healthcare monitoring
* Industrial equipment tracking

---

### 4. Scalability

Cloud computing can easily increase storage and computing resources as the number of IoT devices grows.

#### Benefits

* Flexible infrastructure
* Handles millions of connected devices
* Easy system expansion

---

### 5. Security

Cloud platforms provide security mechanisms to protect IoT systems.

#### Security Features

* Data encryption
* Authentication
* Access control
* Data backup

---

### 6. Cost Efficiency

Organizations can reduce hardware and maintenance costs by using cloud services.

---
### AWS EC2 (Elastic Compute Cloud)

Amazon Web Services EC2 (Elastic Compute Cloud) is a cloud service that lets you rent virtual servers (“instances”) on demand instead of buying physical hardware.

#### Core EC2 Concepts
   
   * Instance — A virtual server running in AWS
   * AMI (Amazon Machine Image) — A preconfigured OS template
 * Instance Type — Defines CPU, RAM, GPU, and performance
* EBS Volume — Persistent storage attached to instances
* Security Group — Virtual firewall controlling traffic
* Key Pair — SSH credentials for secure login
* Elastic IP — Static public IP address
---
### Nginx Web Server

Nginx is a high-performance, open-source web server and reverse proxy known for:
- *High Performance*: Handles multiple concurrent connections efficiently
- *Low Resource Usage*: Consumes minimal memory and CPU resources
- *Reliability*: Proven stability and uptime in production environments
- *Flexibility*: Can serve static content, proxy requests, and load balance traffic
- *Easy Configuration*: Simple configuration syntax for web server setup
